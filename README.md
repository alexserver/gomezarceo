# gomezarceo
www.gomezarceo.mx hugo content


## Prerrequisites

### Install Hugo

- MacOs

```
brew install hugo
```

[installation docs](https://gohugo.io/installation/)


## Quickstart

### Run Hugo Web Server

```
$hugo server
```

### Build Web Site

```
$hugo
```

## Create new content

```
$hugo new content [path]/document.md
```
